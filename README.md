💱 Currency Converter Tool
This project is a web-based Currency Converter built using HTML, CSS, and JavaScript. It enables users to quickly convert money from one currency to another with ease.
🚀 Features
Convert between multiple currencies 
Clean and responsive user interface
Easy-to-use input and dropdown selection
Instant conversion on button click
Beginner-friendly project structure
🛠️ Technologies Used
HTML5
CSS3
JavaScript 
📌 How It Works
Enter the amount you want to convert
Select the From Currency
Select the To Currency
Click on Convert Currency
View the converted result instantly
